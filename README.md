# Mapa_saude-0DS3

## Sobre o Projeto
Este projeto consiste em uma aplicação de software desenvolvida para a disciplina de Engenharia de Software, do curso de Ciência de Dados e Inteligência Artificial, da PUC Minas.
O objetivo é criar um Dashboard Interativo, focado no processamento e visualização de dados públicos de saúde, com base nos Objetivos de Desenvolvimento Sustentável (ODS) da ONU.

**ODS 3 - Saúde e Bem-Estar.**

##  Definição dos Objetivos
* **O Problema:** A carência no acesso e na interpretação de dados públicos da saúde por parte de cidadãos e políticos, especificamente na correlação entre sedentarismo, doenças crônicas e a prática de atividades físicas

* **A Meta:** Um Dashboard de dados interativo desenvolvido em Python, atuando como um sistema que coleta bases de dados abertas, processa as informações e gera visualizações (KPIs e gráficos) que facilitam a tomada de decisão e a conscientização sobre saúde de forma preventiva.

## Repositório
* `/docs`: Contém a documentação de Engenharia de Software (Requisitos, Diagramas, etc).
* `/src`: Contém o código fonte da aplicação (Backend e Frontend).
